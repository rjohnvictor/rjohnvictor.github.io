Portfolio has 3 distinct visual modes, all sharing same content/routing/components:

**Professional** — Default. Recruiters/hiring managers. Clean dark, blue accent, subtle effects.
**Power** — Anime/energy inspired. Ki cursor trail, DBZ power level intro, electric blue/purple. "This is what inspires me."
**Zen** — Reading-focused. Serif font, warm gray, minimal motion, no GPU. "This is how I think."

User values:
- Each mode = different personality aspect, not just a color swap
- Components never know which theme — they receive tokens
- Smooth transitions between modes (no reload)
- Power mode: 60fps, GPU optimized, lazy loaded
- Zen mode: 100 Lighthouse, no particle systems
- Professional mode: 95+ Lighthouse

Future sections user mentioned: Blog, Dashboard (beyond current portfolio sections).

Power Mode theme color inspiration:
- Background: Black
- Primary: Electric Blue, Purple, Cyan
- Small Gold accents
- Everything "feels alive" — energy particles, mouse trails, shockwaves, bloom
